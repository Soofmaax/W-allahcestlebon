# Contribuer

## Pré-requis
- Node.js 20+

## Lancer en local

```bash
npm install
npm run dev
```

## Lints / Debug

```bash
npm run lint:html
npm run lint:css
```

## Déploiement
- Preview Netlify à chaque Pull Request (via GitHub Actions)
- Production Netlify à chaque push sur `main`

Les secrets GitHub nécessaires sont décrits dans `readme.md`.
